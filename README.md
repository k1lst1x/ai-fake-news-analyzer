# ai-fake-news-analyzer
 AI Fake News Analyzer — это веб-платформа для автоматического анализа новостей и определения вероятности того, являются ли они ложными или достоверными.  Проект основан на машинном обучении и обработке естественного языка (NLP). Система обучается на реальных датасетах новостей (включая ISOT Fake News Dataset) и со временем может улучшать качество предсказаний.
